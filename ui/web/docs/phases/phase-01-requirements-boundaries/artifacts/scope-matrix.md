# Scope Matrix

## In Scope (v1)
- Tool 执行步骤卡片基础展示（title/status/time）。
- 可展开查看 `description/code/output`。
- 运行中、成功、等待、失败四类状态可视化。
- 与现有消息列表共存，不破坏普通文本消息。

## Out of Scope (v1)
- 自定义排序、分组、筛选器。
- 跨 turn 的全局工具时间线视图。
- 导出步骤执行报告。
- 复杂动画与可配置主题系统。

## 决策结果（2026-03-09）
- 审批动作在 v1 保持独立 ApprovalPanel，不并入步骤卡片操作区。
- 本次 Tool 卡片接入任务不包含失败步骤一键重试入口。
