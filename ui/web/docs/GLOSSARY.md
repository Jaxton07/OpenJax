# 术语表

- Tool 卡片（Tool Step Card）: 在 assistant 消息上下文中展示单个工具步骤的可视化卡片。
- Step: 工具执行流程中的一个原子步骤（例如 think/tool/shell/approval）。
- StreamEvent: Gateway 推送的流式事件。
- 结构化消息（Structured Message）: 含有类型化字段的消息对象，不仅是文本 content。
- 降级渲染（Fallback）: 在结构化数据不可用时回退到纯文本展示。
- 验收标准（Acceptance Criteria）: 判定阶段是否可结束的硬性条件。
- 证据（Evidence）: 可追溯的客观完成证明。
