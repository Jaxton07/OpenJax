# 01 Test Plan and Acceptance

## 测试分层

- 协议测试：接口字段、错误结构、事件顺序。
- 集成测试：会话创建、提问、审批、关闭全链路。
- 回归测试：重连恢复、限流、超时、幂等。

## 核心验收场景

1. 正常对话流程：start -> stream -> submit -> complete -> shutdown。
2. 工具调用与审批：出现 `approval_requested` 并成功 resolve。
3. 异常流程：认证失败、参数错误、限流、上游不可用。
4. 流式恢复：SSE 断开后按 `event_seq` 正确续读。

## 验收标准

- 协议字段与文档完全一致。
- 无未分类错误码。
- P95 延迟与错误率达到阶段目标（数值由实现阶段填入）。
