# 07 验收与检查清单

## 文档验收
- [x] 目标与范围文档齐全且术语一致。
- [x] 架构、协议、调度、迁移文档可独立评审。
- [x] 分步开发任务具备可执行顺序与依赖说明。

## Core 验收
- [x] 默认路径为真实流式输出。
- [x] 支持单轮多工具并发与依赖调度。
- [x] 工具结果按 `tool_call_id` 结构化回注。
- [x] 失败/超时/取消/审批语义完整。

## Gateway 验收
- [x] 事件协议 v2 可用并兼容 v1。
- [x] 缺口检测与重放机制生效。
- [x] 无静默丢片，终态事件清晰。

## Web 验收
- [x] 高频 delta 下渲染稳定不卡顿。
- [x] 工具批次生命周期展示完整。
- [x] 断线恢复后文本与工具状态一致。

## 指标验收
- [x] TTFT 达到目标区间并优于旧路径。
- [x] Delta 间隔分布稳定。
- [x] 恢复成功率与错误率达到阈值。
