# Rust 内核 + Python 外层重构计划索引

本目录用于集中管理本次架构重构的计划、前置准备、执行清单与阶段产物。

## 文档列表

- `phase-plan-and-todo.md`
  - 本次重构主计划（按阶段）
  - 前置准备清单（可勾选）
  - 开工条件（Go/No-Go）
  - 每阶段验收标准
- `current-handoff-summary.md`
  - 当前节点交接摘要（做了什么、到哪一步、下一步建议）
  - 关键目录与验证命令速查

## 使用约定

1. 先完成“前置准备阶段”再进入代码重构。
2. 每完成一个任务项，直接在清单中勾选并补充日期。
3. 若范围变化，先更新计划文档再执行代码变更。
4. 所有新增重构文档统一放在 `docs/plan/refactor/`。
